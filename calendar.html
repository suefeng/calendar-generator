Printing instructions

  1. ctrl or cmd + p or choose "File" → "Print" from your browser's menu.
  2. View the preview to make sure no parts are cut off. Make sure you have these settings:
    1. "Orientation" is "Landscape"
    2. "Scale" is set to "Fit to page width"
    3. "Margins" is set to "none"
    4. "Print headers and footers" is disabled
    5. "Print background" is enabled
  3. Save as PDF and print the PDF or print directly. If you choose to print directly, remember to try just one page first.
